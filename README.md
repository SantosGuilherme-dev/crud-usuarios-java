# CRUD de Usuários em Java (Console)

Este é um projeto simples de **CRUD (Create, Read, Update, Delete)** desenvolvido em Java puro, utilizando execução via console. O objetivo principal é praticar conceitos fundamentais de Programação Orientada a Objetos (POO) e manipulação de listas.

---

## Funcionalidades

O sistema permite:

- Cadastrar usuários
-  Listar usuários cadastrados
-  Atualizar dados de um usuário
-  Remover usuários
-  Interação via menu no console

---

## Conceitos aplicados

Durante o desenvolvimento deste projeto, foram praticados os seguintes conceitos:

- Programação Orientada a Objetos (POO)
- Encapsulamento
- Classes e objetos
- List (ArrayList)
- Estrutura de controle (switch, loops)
- Organização em camadas simples (Model e Service)
- Entrada de dados com Scanner

---

## Como executar o projeto
1. Clone este repositório no seu computador.
2. Abra a sua IDE (Eclipse ou de sua preferência).
3. Importe o projeto como um projeto Java existente.
4. Execute a classe `Main.java` para iniciar a aplicação via console.
